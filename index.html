Quarto wird in einer sogennanten Markup Sprache geschrieben, die sich Markdown nennt.

(Hier hat sich jemand ein etwas verwirrendes Wortspiel erlaubt)

Lasst uns an dieser Stelle etwas über Markdown sprechen.

  • Eine Markup-Sprache ist eine maschinenlesbare Sprache für die Gliederung und Formatierung von Texten.
  • HTML ist die bekannteste und am häufigsten verwendete Markup Sprache
  • Markdown ist viel simpler in der Schreibweise und sowohl für Mensch und Maschine lesbar
  • Beispielsweise ist der folgende Satz in Markdown geschrieben:
					
							
							# Markdown, eine Markup Sprache

							Dieser Satz ist *wirklich* in **Markdown** geschrieben.
  • Dieser "Quellcode" ist gut lesbar (es ist erkennbar, dass "wirklich" und "Markdown" hervorgehoben werden).
  • Ein Sternchen gilt als Kusive hervorhebung, zwei Sternchen als Fettdruck. Weitere Infos zum Markdown-Syntax findet ihr auf der Quarto Website
  • Ein Programm wie Quarto kann Markdown in PDF oder HTML konvertiert
  • Markdown kann als kostenloser Ersatz für herkömmliche Text-Layout-Systeme wie Microsoft Word genutzt werden
